# Skills Guide

Highlight your technical and soft skills relevant to the job.

## Categories (Recommended for Tech)
- **Languages**: Python, Java, C++, JavaScript.
- **Frameworks**: React, Django, Spring Boot.
- **Tools/DevOps**: Git, Docker, Kubernetes, AWS.
- **Databases**: PostgreSQL, MongoDB.

## Best Practices
- **Relevance**: Tailor skills to the job description (ATS optimization).
- **Honesty**: Don't list skills you can't use in an interview.
- **Soft Skills**: Communication, Leadership (better demonstrated through Experience bullets than a list).
