# Certifications Guide

Professional certifications validate your expertise.

## What to Include
- Industry-recognized certificates (AWS, Cisco, PMP, CPA).
- Language proficiency tests (IELTS, TOEFL, JLPT).

## Format
- **Name**: Full certification title.
- **Issuer**: Organization (e.g., Amazon Web Services).
- **Date**: Date earned (and expiration if applicable).

## Example
> **AWS Certified Solutions Architect – Associate**
> Amazon Web Services | Valid until 2026
